# ChatBot_with_ParlAI

## 1. Description
This is a chat bot based on the ParlAI by Facebook. You can communicate to some chat bots by keyboard and/or voice.

## 2. Operational Environment
- Windows 10 64-bit
- Visual Studio Code
- Python 3.9.4 64-bit
- Browser: Microsoft Edge or Google Chrome
